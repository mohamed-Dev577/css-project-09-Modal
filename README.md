# 🪟 Modal Popup باستخدام CSS

مشروع Front-End بسيط لعرض نافذة منبثقة (Modal) باستخدام HTML و CSS فقط بدون JavaScript.

هدف المشروع هو فهم خصائص:
- position
- fixed
- absolute
- z-index
- التحكم في الطبقات
- إنشاء Overlay (خلفية معتمة)

---

## 🚀 فكرة المشروع

المشروع عبارة عن:

- نافذة منبثقة في منتصف الشاشة
- طبقة خلفية سوداء شفافة (Overlay)
- زر داخل النافذة
- توسيط احترافي في منتصف الشاشة

- الفرق بين relative و absolute و fixed
- كيفية تثبيت عنصر في الشاشة
- كيفية إنشاء طبقة تغطية للخلفية
- مفهوم الطبقات وترتيب العناصر
- التوسيط باستخدام Flexbox
